# nettop
a simple command line Bandwidth Monitor by parsing `/proc/net/dev` to calculator.

## Build
```
$ go build -o nettop nettop.go
```

## Usage
```
Usage of nettop:
	-c uint
		count (0 == unlimit)
	-i string
		interface (default "*")
	-t float
		update time(s) (default 2)
	-v int
		verbosity (default 2)

```

## Output
```
$ ./nettop -i eth0 -t 1 -c 3
BW:	eth0	66.00 B/s	66.00 B/s
```
